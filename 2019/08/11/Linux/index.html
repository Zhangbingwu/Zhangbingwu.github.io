Linux

简介

  • 1.所谓的linux都是统称，具体版本都叫做发行版。
  1. red hat ：目前全球最大的linux发行厂商（收费）
  2. ubuntu：免费，界面好。
  3. 麒麟
  4. cent os 免费，redhat 分支。
  • 2.定位
  1. 操作系统.
  2. 适合于服务器系统,不适合办公系统.
  • 3.linux 相对 window 优点
  1. 稳定.
  2. 安全性比较高
  3. 免费.

系统目录介绍

  1. / 根目录
  2. ~ /root
  3. etc 存放系统配置目录
  4. home 除了root以为所有用户默认在home下新建一个以用户名做为文件夹名称的文件夹。 用户 xxx 对/home/xxx具有完全操作权限.
  5. root 用户root单独文件夹
  6. usr 所有用户安装的软件都放入到这个文件夹中
    在usr/local下新建一个tmp,所有压缩包都上传到tmp中.

常用命令

  1. pwd 打印工作目录，当前所有文件夹全路径
  2. cd 进入文件夹
  3. cd .. 退出当前文件夹
  4. mkdir 新建空文件夹
  5. ls 平铺显示当前文件夹内容 ll 显示详细信息
  6. vi 和 vim 都是编辑文件
    如果文件不存在,带有创建文件功能
    vi 普通编辑 vim 高级编辑(带有颜色)
    进入到编辑器后非可编辑状态,键盘insert启用编辑状态
  7. ：wq 保存并退出
    ：q 退出
    ：q! 强制退出
  8. touch 创建空文件
  9. cat 在命令行中查看文件全部内容
  10. head [-n] 文件名 查看文件前n行，默认10行
  11. tail [-n] 文件名 查看文件后n行，默认10行
  12. tailf 动态显示文件(tomcat日志文件)
  13. echo ‘内容’ >> 文件名 向文件中添加一些内容
  14. ifconfig 打印网卡信息，查询当前ip
  15. reboot 重启
  16. tar zxvf 文件名 解压缩
    v 解压过程显示
  17. zip unzip 压缩 解压
  18. cp [-r] 原文件 新文件路径 -复制文件，复制文件夹需要加-r
  19. ctrl+c 中断
  20. mv 原文件 新文件 -剪切，移动 可以用来修改文件名字
  21. rm [-r][-f] 文件名 -移除文件，加-r移除目录 ，加-f不提示直接移除，-rf
  22. clear 清屏

不同用户登录

root用户 显示 #
普通用户 显示 $

Xshell

  1. linux 客户端工具.
  2. CentOS 中默认只开启 22 端口.其他端口都被防火墙拦截.

Filezilla

  • 1.FTP协议
  1. (FileTransfer Protocol）文件传输协议
  2. 作用:互联网中,计算机之间传输文件需要遵守的协议.
  • 2.fileziila
  1. 基于 FTP 协议的客户端工具.
  2. 使用 filezilla 可以快速完成文件传输功能.

修改环境变量，防火墙

修改环境变量

  • 1.# vim/etc/profile
  • 2.文件内容
  1. export 添加或修改变量
  2. 多个变量之间 : 隔开，不像windows ;
  3. $变量名 表示引用
  1. 修改后需要解析文件(或重启系统)

    source /etc/profile

防火墙

  1. vim /etc/sysconfig/iptables

  2. 8080:9000 表示从8080到9000全放行
  3. 修改后重启防火墙

    service iptables restart

启动tomcat

  • linux中 .sh 表示可执行文件
  • ./startup.sh 启动tomcat
  • ./startup.sh & tailf ../logs/catalina.out 启动tomcat并且显示日志
