Posted on 2019-08-08 Edited on 2019-08-09 In 教学

JSP九大内置对象复习

  • 九大内置对象是JSP转成java文件后，在局部变量为我们定义好的。
  1. page 当前页面
  2. pageContext 封装了其他八个内置对象
  3. request 封装了请求的所有信息
  4. response 封装了响应的所有信息
  5. Exception 异常对象
  6. out resp.getWriter()获得的用于输出字符流的对象
  7. session req.getSession() 封装了所有会话的信息
  8. application 所有信息 getServletContext();req.getServletContext;
  9. config 配置信息

四大作用域对象

  1. page 当前页面内不会重新实例化。
  2. request 一次请求中不会重新实例化。
  3. session 一次会话中不会重新实例化。
    1.客户端中Cookie保留的JSESSIONID和服务器中一致，session就不会重新实例化。(在默认时间内)
    浏览器关闭（cookie失效）
默认时间内，没有交互。
    2.默认时间在web.xml中修改，<session-config>标签。
  4. application 服务器启动时实例化，服务器关闭销毁。

SpringMVC作用域传值

1.原生Servlet

2.Map传值

  • 把 map 中内容放在 request 作用域中
    ${requestScope.map}

  • spring 会对 map 集合通过 BindingAwareModelMap 进行实例化

  • 也可以放对象，集合。

class org.springframework.validation.support.BindingAwareModelMap

3.Model

  • SpringMVC 中 Model 接口
  • 把内容最终放入到 request 作用域中

4.ModelAndView

  • SpringMVC 中 ModelAndView 类
  • 把内容最终放入到 request 作用域中

文件的上传与下载

文件下载

  1. 设置响应头中Content-Disposition为attachment 文件为下载类型
    fileName就是下载时显示的文件名称

    response.setHeader("Content-Disposition","attachment;fileName="+fileName);

    浏览器默认为 inline 能显示就显示，显示不了就下载。

  2. 获得文件夹的绝对路径
    String path = request.getServletContext().getRealPath("files");

  3. 将文件转为二进制流
    byte[] bytes = FileUtils.readFileToByteArray(new File(path, fileName));

  4. 获得响应输出流并写出

    注意：需要在springmvc中放行静态资源files
    <mvc:resources location="/files/" mapping="/files/**"/>

文件上传

  • 基于 apache 的 commons-fileupload.jar 完成文件上传.
  • MultipartResovler 作用:
    把客户端上传的文件流转换成 MutipartFile 封装类.
    通过 MutipartFile 封装类获取到文件流
  • 表单数据类型分类
    在<form>的 enctype 属性控制表单类型
  1. 默认值 application/x-www-form-urlencoded,普通表单数据.(少
    量文字信息)
  2. text/plain 大文字量时使用的类型.邮件,论文
  3. multipart/form-data 表单中包含二进制文件内容.
  • 实现步骤:

  1. 导入 springmvc 包和 apache 文件上传 commons-fileupload 和
    commons-io 两个 jar

  2. 编写 JSP 页面
    上传: <input type="file" name="file">

    注意：这里的name的值，必须和 MultipartFile 的参数名称相同

  3. 配置 springmvc.xml
    MutipartResovler解析器

    异常解析器
    private Properties exceptionMappings;
    对Properties类型设置注入。
    <prop key=异常全称>跳转的页面</prop>

  4. 编写控制器controller
